# template-vuepress

> Create a VuePres project with [SAO](https://github.com/egoist/sao).

## Usage

Install [SAO](https://github.com/egoist/sao) first.

```bash
yarn global add sao
# or
npm i -g sao
```

### From npm

```bash
sao vuepress my-project
```

### From git

```bash
sao haolchen/template-vuepress my-project
```
